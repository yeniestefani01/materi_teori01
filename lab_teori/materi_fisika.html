Selamat Datang di Program Fisika Teori dan Komputasi

Program ini memadukan pemahaman mendalam tentang teori fisika dengan penerapan komputasi untuk memecahkan masalah fisika yang kompleks. Mahasiswa akan mempelajari konsep-konsep fisika teoritis yang mendalam serta teknik komputasi modern untuk mensimulasikan dan menganalisis fenomena fisika.

FOKUS PEMINATAN

Condensed Matter Physics

Software simulasi material (VASP, Quantum ESPRESSO), tools analisis struktur kristal, dan lingkungan komputasi Linux untuk studi fisika bahan mampat dan teori perubahan fase.

Cosmology

Kluster komputasi kinerja tinggi untuk simulasi kosmologis, software pemodelan relativitas umum, dan tools analisis dinamika nonlinier untuk studi evolusi alam semesta.

Particle Physics

Infrastruktur komputasi untuk simulasi tumbukan partikel, tools analisis data eksperimen, dan software untuk studi mekanika kuantum lanjutan dan dinamika nonlinier.

Computation

Laboratorium pemrograman lengkap dengan framework machine learning, tools pengembangan web, environment Linux, dan software untuk analisis dinamika nonlinier.

Nuclear Physics

Software simulasi reaksi nuklir, tools untuk aplikasi kedokteran nuklir, dan lingkungan komputasi untuk studi fisika inti dan aplikasi medis.

MISI KAMI

Mengembangkan kemampuan analitis dan komputasi mahasiswa dalam memahami dan memecahkan masalah fisika yang kompleks, serta mempersiapkan mereka untuk karir dalam penelitian, industri, dan pendidikan fisika tingkat lanjut.
